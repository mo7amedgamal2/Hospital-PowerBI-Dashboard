# 🏥 Hospital Data Warehouse & BI Solution

An end-to-end **Business Intelligence** project that demonstrates the complete data warehousing process using **SQL Server**, **SSIS**, and **Power BI**.

The project transforms healthcare operational data into a centralized analytical warehouse and delivers interactive dashboards for business reporting and decision-making.

---

## 📌 Project Overview

This project implements a Healthcare Data Warehouse solution that integrates transactional healthcare data into a dimensional model for analytical reporting.

The solution includes database design, ETL development, data warehouse implementation, and interactive Power BI dashboards.

## 📂 Data Warehouse Architecture

The project follows a modern Business Intelligence architecture:

```
OLTP Database
      │
      ▼
ETL Process (SSIS)
      │
      ▼
Healthcare Data Warehouse
      │
      ▼
Power BI Dashboard
```

---

## ⚙️ Project Workflow

### 1. Database Design

- Created healthcare transactional databases.
- Designed relational database tables.
- Defined primary and foreign key relationships.

---

### 2. Data Warehouse

- Designed a Star Schema.
- Created Fact and Dimension tables.
- Optimized the warehouse for analytical reporting.

---

### 3. ETL Development

Developed SSIS packages to:

- Extract healthcare data.
- Transform and clean data.
- Load data into the warehouse.

---

### 4. Power BI Dashboard

Developed interactive dashboards to analyze:

- Claims
- Encounters
- Healthcare Costs
- Payers
- Healthcare KPIs
